# **App Name**: Zapizza PWA

## Core Features:

- Mobile OTP Login: User login via mobile number and OTP using Firebase Authentication.
- City Selection: Allows customers to select their city or auto-detect location for menu filtering.
- Menu Display: Displays menu based on selected city, with category-based scrolling.
- Real-time Order Tracking: Tracks orders in real-time with status updates and notifications sent to the customer via FCM.
- Razorpay Payment: Integrates Razorpay for secure payment processing with order confirmation.
- Outlet Admin Panel: Provides a panel for outlet admins to manage orders, toggle item availability, and view daily sales.
- Super Admin Dashboard: A central dashboard for the franchise owner to view all cities, outlets, orders, revenue, and manage menus, powered by Firestore.

## Style Guidelines:

- Primary color: Vibrant red (#E53935), reminiscent of classic pizzeria branding, conveying appetite and excitement.
- Background color: Light gray (#F5F5F5) provides a clean and modern canvas that avoids distracting from the food imagery.
- Accent color: Mustard yellow (#FDD835), a contrasting hue, emphasizes key interactive elements, giving the design an engaging visual punch.
- Body text: 'PT Sans' for readability, for menu descriptions and order details.
- Headline font: 'Space Grotesk' for a clean, modern and playful look, especially for category headers and titles.
- Use consistent, clear icons for menu categories, order status, and account functions.
- Subtle transitions for loading states and order status updates.
